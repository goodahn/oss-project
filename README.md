# oss-project
# get docker image
```
docker pull goodahn/for_arcus
docker pull goodahn/for_hubble
```
# start arcus
```
./run_arcus.sh
su - john
cd arcus/scripts
sh arcus-start.sh
```
# start hubble
```
./run_hubble.sh
cd hubblemon
./hubble_start.sh
```
# start simple web system
```
pip3 install -r requirement.txt
./routes.py
```
